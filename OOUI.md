# OOUI オブジェクトモデル定義

## 用語定義

| 用語 | 定義 |
|------|------|
| 補給 | サプリを飼い主判断で与える記録 |
| 投与 | くすりを処方に基づいて与える記録 |
| お世話 | 日報に紐づく日々のケア記録の総称 |
| コレクション | 同種のオブジェクトの一覧画面 |
| シングル | 個別オブジェクトの詳細画面 |

## オブジェクトモデル

### メインオブジェクト

#### 猫
- 写真
- 名前
- 性別
- 生年月日（推定）
- 家族になった日
- 毛色
- マイクロチップ
- AIPO登録番号
- 疾患
- 疾患の編集
- 写真の編集

### オブジェクト

#### 日報
- 年月日
- 日付を選ぶ

#### お世話
- 種類：食事・うんち・おしっこ・ゲロ・体重・補給・投与・受診
- TP
- …（その他検査項目）
- 検査写真
- 血液検査メモ
- 血液検査の編集

#### 尿検査結果
- 尿比重
- pH
- たんぱく
- 潜血
- …（その他検査項目）
- 尿検査写真
- 尿検査メモ
- 尿検査の編集

### エンティティオブジェクト

#### フード
- フード画像
- メーカー名
- 種類
- 商品名
- 単位：g／袋／本／粒
- 栄養成分
  - kcal
  - リン
  - タンパク質
  - マグネシウム
  - カルシウム
- フード新規追加
- フードの編集

#### サプリ（エンティティ・マスター未確定）
※ マスター管理画面は未確定。
　 フードタブ内に含める案とサプリマスターを
　 独立して作る案を検討中。
- サプリの名前
- サプリの効能
- サプリの量
- サプリのメモ
- 編集

#### くすり（エンティティ・マスター未確定）
※ マスター管理画面は未確定。処方オブジェクトからの
　 連携を検討中。専門家と実装を見ながら決定予定。
- くすりの名前
- くすりの効果
- くすりのメモ
- 編集

#### 動物病院
- 動物病院名
- 住所
- 電話番号
- 診察時間
- 動物病院メモ
- 動物病院の編集

### レポート（読み取り専用オブジェクト）

#### レポート
- 統計
- 種類：食事・うんち・おしっこ・ゲロ・体重・サプリ・くすり・受診

## 画面遷移

### お世話の画面遷移

日報コレクション
　└─ 日報シングル
　　　└─ お世話コレクション
　　　　　├─ 食事シングル
　　　　　├─ うんちシングル
　　　　　├─ おしっこシングル
　　　　　├─ ゲロシングル
　　　　　├─ 体重シングル
　　　　　├─ 補給シングル
　　　　　└─ 投与シングル

### 受診の画面遷移

猫シングル → 受診コレクション
　　　　　　　　　└─ 受診シングル
　　　　　　　　　　　├─ 処方シングル
　　　　　　　　　　　├─ 尿検査シングル
　　　　　　　　　　　└─ 血液検査シングル

### レポートの画面遷移

猫シングル → レポートコレクション
　├─ レポートシングル（うんち）
　├─ レポートシングル（ゲロ）
　├─ レポートシングル（おしっこ）
　├─ レポートシングル（食事）
　├─ レポートシングル（体重）
　├─ レポートシングル（補給）
　├─ レポートシングル（投与）
　└─ レポートシングル（受診）
　　　├─ レポートシングル（処方）
　　　├─ レポートシングル（尿検査）
　　　└─ レポートシングル（血液検査）
